# The Archives

Explore, survive, and build in an infinite procedurally generated world.

One of many side projects, not playable for now ;3

The world generation uses modified Wave Function Collapse algorithm, based on Marian's awesome [Generating an infinite world with the Wave Function Collapse algorithm](https://marian42.de/article/infinite-wfc/) blog post
